PROPOSITIONS UI — Panel Stats / Inventaire (v2)

Principes appliques a toutes les propositions :
1. Pas de barres de vie / barres de stats — uniquement des valeurs numeriques
2. Stats scrollables — le panel peut avoir plus de stats que l'espace visible
3. Tap sur une stat = detail du calcul — ex: "ATK 57 = 45 base + 12 equip + 10% trait"
4. Le style Prop 5 (onglets icones) est le favori

Sommaire

  1. Toggle Classique — 2 onglets horizontaux texte
  2. Bottom Sheet Slide-Up — Panel extensible par swipe
  3. Segmented Control — Pill-style iOS
  4. Swipe Horizontal — Pages gauche/droite
  5. Onglets Icones — Compact, extensible (FAVORI)
  6. Stats Inline + Drawer — Stats fixes, inventaire en tiroir
  7. Card Flip — Carte retournable
  8. Split Panel — Stats + Equip cote a cote
  9. Radial / Mannequin — Slots autour du perso
  10. Mini-Stats + Popup — Compact + detail plein ecran

Detail stat (commun a toutes les props) — Quand le joueur tap une ligne de stat, un dropdown s'ouvre en-dessous montrant la decomposition :
